# Tutedude Assignment 3  
## Module 4: Functions & Modules in Python

This repository contains solutions for Assignment 3 of the Tutedude Python course.  
The assignment focuses on **functions**, **recursion**, and usage of the **math module**.

---

## Task 1: Calculate Factorial Using a Function

### Description
- Defines a function `factorial()` that calculates the factorial of a number.
- Uses **recursion** to compute the result.
- Takes input from the user.
- Prints the calculated factorial.

### File
- `task1_factorial.py`

### Concepts Used
- Functions
- Recursion
- Conditional statements
- User input

---

## Task 2: Using the Math Module for Calculations

### Description
- Takes a number as input from the user.
- Uses the `math` module to calculate:
  - Square root of the number
  - Natural logarithm (base e)
  - Sine of the number (in radians)
- Displays all calculated results.

### File
- `task2_math_module.py`

### Concepts Used
- `math` module
- Built-in mathematical functions
- User input
