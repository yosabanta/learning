# Tutedude Assignment 2 – Control Structures in Python

## Overview
This assignment demonstrates the use of Python control structures such as
conditional statements and loops.

It contains two tasks implemented using basic Python concepts.

---

## Task 1: Check Even or Odd

**File:** `task1_even_odd.py`

### Description
- Takes an integer input from the user.
- Uses an `if-else` statement to check whether the number is even or odd.
- Displays the result accordingly.

### Concepts Used
- User input
- Type conversion (`int`)
- Conditional statements (`if-else`)
- Modulus operator (`%`)

---

## Task 2: Sum of Numbers from 1 to 50

**File:** `task2_sum_1_to_50.py`

### Description
- Uses a `for` loop to iterate from 1 to 50.
- Calculates the sum of all numbers in this range.
- Prints the final sum.

### Concepts Used
- `for` loop
- `range()` function
- Accumulator variable
