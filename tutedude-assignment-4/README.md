# Tutedude Assignment 4 – Files and Exception Handling in Python

## Description
This assignment demonstrates file handling and exception handling in Python.
It includes reading from a file, handling errors, and writing/appending data to a file.

---

## Task 1: Read a File and Handle Errors
- Reads content from `sample.txt`.
- Prints the file content line by line.
- Handles the `FileNotFoundError` if the file does not exist.

**File:** `task1_read_file.py`

---

## Task 2: Write and Append Data to a File
- Takes user input and writes it to `output.txt`.
- Appends additional user input to the same file.
- Reads and displays the final content of the file.

**File:** `task2_write_append_file.py`

---

## How to Run

Run Task 1:
```bash
python task1_read_file.py
